🎓 College Event Management System

A web-based College Event Management System developed using PHP and MySQL. This project helps colleges manage events, announcements, registrations, and student participation through an easy-to-use interface.

📌 Features

- 👤 User Login System
- 🔒 Change Password Functionality
- 📢 Event Announcements
- ➕ Create New Events
- 📅 Manage College Events
- 🖼️ Event Gallery
- 📊 Dashboard for Event Management
- 💾 Database Integration using MySQL

🛠️ Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP
